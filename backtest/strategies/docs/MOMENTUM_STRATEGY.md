# 모멘텀 전략 V2 — 승율 개선 가이드

## 전략 개요

모멘텀(Momentum) 기반의 추세 추종 전략. N일 수익률이 양수면 보유, 음수면 현금으로 전환. 가격 추세의 지속성을 활용한다.

**파일**: `backtest/strategies/momentum.py`  
**클래스**: `MomentumStrategy`

---

## V1 → V2 개선사항

| 항목 | V1 (기존) | V2 (개선) |
|------|----------|----------|
| 모멘텀 측정 | 장기 단일 (120일) | **듀얼 모멘텀** (장기 + 단기 모두 양수) |
| 리밸런싱 사이 손절 | 없음 (20일 동안 무방비) | **ATR 트레일링 스톱** (리밸런싱 사이 보호) |
| 추세 필터 | 없음 | **MA200 필터** (하락장 자동 청산) |
| 추세 이탈 청산 | 없음 | **즉시 청산** (리밸런싱 기다리지 않음) |
| 포지션 동기화 | 없음 | **외부 청산 반영** |

---

## 핵심 개선: 듀얼 모멘텀

### 기존 방식의 문제
```
120일 수익률 > 0% → 매수
```
- 장기 모멘텀은 양수이나 최근 하락 중인 종목 진입
- 고점 부근에서 늦은 진입 후 하락 손실

### 개선된 방식
```
장기 모멘텀(120일) > 0% AND 단기 모멘텀(20일) > 0% → 매수
```
- **장기 모멘텀**: 시장 방향성 확인 (추세 존재 여부)
- **단기 모멘텀**: 최근 가속도 확인 (지금도 오르고 있는지)
- 두 가지 모두 양수일 때만 진입 → 늦은 진입 방지

### 왜 효과적인가
```
장기 +, 단기 + → 추세 중간 진입 (이상적)
장기 +, 단기 - → 추세 후반 또는 조정 (진입 보류)
장기 -, 단기 + → 반등 시작? (불확실, 진입 보류)
장기 -, 단기 - → 명백한 하락 (현금 보유)
```

---

## 핵심 개선: 리밸런싱 사이 ATR 트레일링 스톱

### 기존 문제
```
리밸런싱 주기 = 20일
→ 20일 동안 급락해도 손절 불가
```

### 개선
```
매 봉마다 ATR 트레일링 스톱 업데이트
→ 리밸런싱 시점과 무관하게 급락 즉시 손절
```

---

## 파라미터

| 파라미터 | 기본값 | 설명 |
|---------|--------|------|
| `lookback` | 120 | 장기 모멘텀 측정 기간 (거래일) |
| `short_lookback` | 20 | 단기 모멘텀 측정 기간 (0=비활성화) |
| `min_momentum` | 0.0 | 장기 모멘텀 최소 기준 (%) |
| `invest_pct` | 1.0 | 투자 비율 |
| `rebalance_freq` | 20 | 리밸런싱 주기 (거래일, 약 1개월) |
| `trend_window` | 200 | 추세 필터 MA 기간 (0=비활성화) |
| `trail_mult` | 3.0 | ATR 트레일링 스톱 배수 |
| `market_df` | None | 시장 레짐 필터 |

---

## 진입 조건

**리밸런싱 시점에서 모두 충족해야 매수:**

1. **장기 모멘텀**: `(현재가 / 120일 전 가격 - 1) × 100 > min_momentum(0%)`
2. **단기 모멘텀**: `(현재가 / 20일 전 가격 - 1) × 100 > 0%`
3. **추세 필터**: `가격 > MA(trend_window=200)`
4. **레짐 필터**: `시장 > MA200` (market_df 제공 시)

---

## 청산 조건

### 즉시 청산 (리밸런싱 무관)
- `가격 < MA(trend_window)` → 추세 이탈 즉시 청산
- ATR 트레일링 스톱 트리거 → 급락 손절
- 시장 레짐 이탈

### 리밸런싱 시점 청산
- 위 진입 조건 중 하나라도 불충족 시 청산

---

## 모멘텀 vs 듀얼 모멘텀 비교

```
종목: 상반기 강세 → 하반기 조정 케이스

단일 모멘텀 (120일):
  6월: 120일 수익률 +30% → 매수
  7월: 120일 수익률 +20% → 유지 (조정 시작인지 모름)
  8월: 120일 수익률 +5%  → 유지 (계속 조정 중)
  9월: 120일 수익률 -2%  → 매도 (너무 늦게 손절)

듀얼 모멘텀 (120일 + 20일):
  6월: 120일 +30%, 20일 +5%  → 매수 ✓
  7월: 120일 +20%, 20일 -3%  → 청산! (조정 감지)
  8월: (관망)
  9월: (관망)
```

## 사용 예시

```python
from backtest.strategies import MomentumStrategy

# 듀얼 모멘텀 + 손절 강화 (권장)
strategy = MomentumStrategy(
    ticker="005930",
    lookback=120,         # 장기: 6개월
    short_lookback=20,    # 단기: 1개월
    trend_window=200,     # MA200 추세 필터
    trail_mult=3.0,       # ATR × 3 트레일링 스톱
    rebalance_freq=20,    # 월 1회 리밸런싱
    invest_pct=1.0,
)

# 보수적 설정 (포지션 50%, 강한 모멘텀만)
strategy = MomentumStrategy(
    ticker="005930",
    lookback=120,
    short_lookback=20,
    min_momentum=5.0,     # 6개월 5% 이상 상승 시만 진입
    trend_window=200,
    trail_mult=3.0,
    invest_pct=0.5,
)
```

## 적합한 종목

- **추세가 명확한 성장주** (추세 지속 기간 길수록 유리)
- 섹터 ETF, 시장 지수 (모멘텀 지속성 높음)
- 분기 실적 발표로 트렌드가 바뀌는 종목

---

## 비적합 상황

- 박스권 횡보 종목 (모멘텀 신호 불안정)
- 뉴스 민감도 높은 소형주 (급격한 추세 반전)
- 변동성 극히 높은 종목 (ATR 스톱 너무 자주 발동)
