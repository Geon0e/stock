# RSI 전략 V2 — 승율 개선 가이드

## 전략 개요

RSI(상대강도지수) 기반의 평균회귀 전략. 과매도 구간에서 매수, 과매수 구간에서 매도하는 역추세 전략.

**파일**: `backtest/strategies/rsi.py`  
**클래스**: `RSIStrategy`

---

## V1 → V2 개선사항

| 항목 | V1 (기존) | V2 (개선) |
|------|----------|----------|
| RSI 계산 | 단순 평균 (부정확) | **Wilder 평활** (표준) |
| 진입 방식 | RSI < 30 터치 시 즉시 매수 | **RSI 크로스오버** (과매도 하회 후 상향 돌파) |
| 추세 필터 | 없음 | **MA50 필터** (상승 추세에서만 진입) |
| 손절 | 없음 (무한 보유) | **ATR 트레일링 스톱** |
| 포지션 동기화 | 없음 | **외부 청산 반영** |
| 레짐 필터 | 없음 | **시장 MA200 옵션** |

---

## 핵심 개선: 크로스오버 진입

### 기존 방식의 문제
```
RSI < 30 → 즉시 매수
```
- 강한 하락추세에서 RSI가 30 아래에 오래 머무름 → 계속 물타기
- "낙하하는 칼날" 문제

### 개선된 방식
```
직전 RSI < 30 AND 현재 RSI >= 30 → 매수 (반등 확인 후 진입)
```
- RSI가 과매도 구간에 진입했다가 **빠져나오는 순간** 진입
- 실제 반등이 시작됨을 확인한 후 진입 → 허위 신호 크게 감소

## 파라미터

| 파라미터 | 기본값 | 설명 |
|---------|--------|------|
| `period` | 14 | RSI 계산 기간 |
| `oversold` | 30.0 | 과매도 기준 (이하 하락 후 상향 크로스 시 매수) |
| `overbought` | 70.0 | 과매수 기준 (이상 시 매도) |
| `invest_pct` | 1.0 | 투자 비율 |
| `trend_window` | 50 | 추세 필터 MA 기간 (0=비활성화) |
| `trail_mult` | 2.0 | ATR 트레일링 스톱 배수 (0=비활성화) |
| `exit_ma_break` | True | MA 이탈 시 청산 여부 |
| `market_df` | None | 시장 레짐 필터용 데이터 |
| `use_vol_sizing` | False | 변동성 조정 포지션 사이징 |
| `risk_pct` | 0.02 | 거래당 리스크 비율 |

---

## 진입 조건

**모두 충족해야 매수:**

1. **RSI 크로스오버**: `직전 RSI < oversold(30)` AND `현재 RSI >= oversold(30)`
2. **추세 필터**: `가격 > MA(trend_window=50)` (상승 추세 컨텍스트)
3. **과매수 아님**: `RSI < overbought(70)` (반등이 이미 과매수 아님 확인)
4. **레짐 필터**: `시장 > MA200` (market_df 제공 시)

---

## 청산 조건

**하나라도 충족 시 매도:**

1. `RSI > overbought(70)` — 이익 실현
2. `가격 < MA(trend_window)` — 추세 이탈 방어
3. ATR 트레일링 스톱 트리거 — 손절
4. 시장 레짐 이탈

---

## 승율 향상 원리

```
허위 신호 제거:
  - 하락추세 중 진입 → MA50 필터로 차단
  - RSI 터치만으로 진입 → 크로스오버 확인으로 차단
  
손실 제한:
  - 기존: 손절 없음 → 큰 손실 가능
  - 개선: ATR 트레일링 스톱 → 손실 자동 제한

수익 보호:
  - 트레일링 스톱이 수익 구간에서 점진적 상향
  - 이익 실현 자동화
```

## 사용 예시

```python
from backtest.strategies import RSIStrategy

strategy = RSIStrategy(
    ticker="005930",
    period=14,
    oversold=30,
    overbought=70,
    trend_window=50,      # MA50 추세 필터
    trail_mult=2.0,       # ATR × 2 트레일링 스톱
    exit_ma_break=True,   # MA 이탈 시 즉시 청산
    invest_pct=1.0,
)
```

## 적합한 종목

- 등락이 명확한 **박스권 + 완만한 우상향** 종목
- 변동성이 높은 대형주 (RSI 신호 빈도 적절)
- **하락추세 종목에는 부적합** (추세 필터 필수)

---

## 비적합 상황

- 강한 단방향 추세 (상승/하락): RSI가 과매수/과매도 구간에 오래 머뭄
- 거래량 극히 적은 종목: 신호 왜곡
- 급격한 뉴스 이벤트: RSI 크로스가 의미 없음
